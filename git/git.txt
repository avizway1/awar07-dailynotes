Branching helps us to create a seperate line of development to work on new features, bug fixes, enhancements without touching your main branch/codebase.

git branch <branch-name>	--> create branch
git switch <branch-name>	--> switching to the branch

git branch -b <branch-name>	--> create branch and switch automatically
